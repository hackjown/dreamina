## 测试建议

### 1. 后端启动测试 ✅
```bash
# 启动后端
node server/index.js

# 健康检查
curl http://localhost:3001/api/health

# 预期输出：{"status":"ok","mode":"direct-jimeng-api"}
```

### 2. API 测试

#### 项目管理 ✅
```bash
# 创建项目
curl -X POST http://localhost:3001/api/projects \
  -H "Content-Type: application/json" \
  -d '{"name":"测试项目","description":"测试"}'

# 获取项目列表
curl http://localhost:3001/api/projects

# 获取项目详情
curl http://localhost:3001/api/projects/1

# 更新项目
curl -X PUT http://localhost:3001/api/projects/1 \
  -H "Content-Type: application/json" \
  -d '{"name":"更新后的项目名"}'

# 删除项目
curl -X DELETE http://localhost:3001/api/projects/1
```

#### 任务管理 ✅
```bash
# 创建任务
curl -X POST http://localhost:3001/api/projects/1/tasks \
  -H "Content-Type: application/json" \
  -d '{"prompt":"测试提示词"}'

# 获取任务列表
curl http://localhost:3001/api/projects/1/tasks

# 获取任务详情
curl http://localhost:3001/api/tasks/1

# 更新任务
curl -X PUT http://localhost:3001/api/tasks/1 \
  -H "Content-Type: application/json" \

# 删除任务
curl -X DELETE http://localhost:3001/api/tasks/1
```

#### 素材管理 ✅
```bash

  -F "images=@/path/to/image1.jpg" \

# 删除任务素材
```

#### 全局设置 ✅
```bash

# 更新全局设置
```
